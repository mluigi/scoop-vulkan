# scoop-vulkan

Custom [Scoop](https://scoop.sh) bucket that extends the official Vulkan SDK manifest with optional development components:

- **volk** — meta-loader for Vulkan
- **VMA** — Vulkan Memory Allocator
- **Debug libraries** — validation layer debug symbols

The manifest is automatically synced from the official [ScoopInstaller/Main](https://github.com/ScoopInstaller/Main) bucket daily and patched to include these extras.

## Usage

```powershell
scoop bucket add scoop-vulkan https://github.com/mluigi/scoop-vulkan
scoop install scoop-vulkan/vulkan
```

To update:

```powershell
scoop update
scoop update vulkan
```

## Why

The official Scoop Vulkan manifest installs the base SDK only. This bucket patches the installer arguments to also select `com.lunarg.vulkan.volk`, `com.lunarg.vulkan.vma`, and `com.lunarg.vulkan.debug` — components that are opt-in during a manual GUI install but not exposed in the upstream manifest.

## Manifest

`bucket/vulkan.json` is auto-generated by a GitHub Action and should not be edited manually. To change which components are included, edit the `jq` patch in [`.github/workflows/sync-vulkan.yml`](.github/workflows/sync-vulkan.yml).
